## 이 프로젝트는 종전에 CMS 툴에 우리은행 라벨관리 화면을 추가한것
단순하게 input 창에서 라벨이름과 메뉴이름을 입력하면 화면에 표시되고 있는 메뉴 이름이 바뀌는 것.

## 설명
처음엔 i18n 기능을 이용해서 구현하려고 했었으나, 해당 기능은 빌드 과정이 필요하여 우리은행의 요구사항과 다름.
따라서 t() 호출시 app.tsx 에서 등록해놓은 JSON 에 있는 라벨을 찾아서 표시하는 방식으로 바뀌며
실질적으로 i18n 은 사용하지 않는다고 봐도 무방하다.


## 사용법
cd ./razor-contentbuilder 
yarn dev

http://localhost:3004/aa 


## 기타 참고사항
* 한승훈 과장 도움으로 만들었으니 기술적인 문제
* 현재 프로젝트는 기존의 NEO-React-contentbuilder 기반으로 만들어졌으며,
이 프로젝트의 의존성 중 우리은행 외부망에서 받아올 수 없는 것이 있어 yarn 으로 설치 불가능함.
따라서 외부에서 의존성을 모두 설치 후 해당 파일까지 포함하여 (노진완 대리의 네오 NAS 이용하여) 옮겨왔음.
